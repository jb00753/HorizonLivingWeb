Horizon Living Website:
A website for a hypothetical assisted living home built as part of a cohort project in my User Centric Computing course. The design prioritized accessibility and usability, refined through multiple rounds of user testing.
Tech Stack: HTML | CSS | JavaScript

Link to website here: [https://github.com/jb00753/HorizonLivingWeb/deployments/github-pages](https://jb00753.github.io/HorizonLivingWeb/)

Problem Statement:
Design an accessible, user-friendly website for a hypothetical assisted living facility that directly addresses concerns raised by a real user test panel.
Approach

Developed initial wireframes as a cohort to establish a shared design vision
Conducted interviews with a panel of users to gather early feedback
Refined wireframes based on findings, then began development
Iterated through approximately 6 rounds of user testing and redesign before reaching the final product

Results:

Accessibility scores improved across iterations
Task completion rates improved drastically between the first and final rounds of testing
Final product reflects direct, evidence-based input from real users
